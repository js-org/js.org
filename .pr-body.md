> The site content is the documentation and live demo for **riyal**, a TypeScript/JavaScript toolkit that ships the official Saudi Riyal currency symbol (Unicode U+20C1, SAMA 2025). The package is published on npm as [`riyal`](https://www.npmjs.com/package/riyal) and provides a web font, CSS, React components, React Native support, a Web Component, a Tailwind CSS plugin, a Next.js font loader helper, OG image cards, and a CLI — all in a single package. It is relevant to JavaScript developers specifically because it solves a concrete, language-level problem in the JS ecosystem: rendering the new Saudi Riyal glyph reliably across modern JS frameworks (React, Next.js, Tailwind, Web Components, React Native) using TypeScript-first APIs, with browser/Node fallbacks and `Intl.NumberFormat` integration. The docs site itself is a Vite + React app that demonstrates each integration interactively.
